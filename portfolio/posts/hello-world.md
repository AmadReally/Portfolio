# Hello World

Welcome to my devlog. This is where I'll document experiments, projects, and things I learned the hard way.

Because this whole portfolio is a terminal. Keeping the blog here felt right — no separate page, no context switch. You're already in the console.

## What to expect

- Project deep-dives
- Quick notes on libraries and tools
- Experiments that worked (and ones that didn't)

Type `blog` to list all posts. Use `blog read <slug>` to open one.
